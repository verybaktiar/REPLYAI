1 Overview

Fitur WhatsApp Inbox memungkinkan admin untuk melihat, memantau, dan membalas pesan WhatsApp langsung dari dashboard ReplyAI. Fitur ini didesain menyerupai tampilan WhatsApp Web untuk kemudahan penggunaan, namun terintegrasi penuh dengan database sistem.

3 Cara Penggunaan

A. Mengakses Inbox

Klik menu WhatsApp di sidebar, lalu pilih submenu Inbox (atau akses URL /whatsapp/inbox).

B. Mencari Kontak

Gunakan kolom pencarian di bagian atas daftar chat. Anda bisa mencari berdasarkan Nama Profil atau Nomor Telepon.

C. Mengirim Pesan Media

  1. Buka percakapan yang dituju.
  2. Klik ikon Attach (üìé) di sebelah kiri kolom ketik.
  3. Pilih file gambar (JPG/PNG) atau dokumen (PDF). Maksimal 10MB.
  4. (Opsional) Tambahkan caption pada kolom teks.
  5. Klik tombol Kirim (‚û§).

4 Teknis & Struktur Kode

Controller: WhatsAppInboxController.php

  • -> index() : Render halaman
  • -> getConversations() : Return JSON list chat (Filtered: No Groups)
  • -> getMessages($phone) : Return JSON riwayat chat
  • -> send() : Handle text & file upload via FormData

Frontend: inbox.blade.php

  • Stack: Alpine.js + Tailwind CSS
  • State: conversations[], messages[], activeChat
  • Polling: setInterval 5000ms untuk fetch data baru

5 Batasan Saat Ini

  • Incoming Media: Pesan gambar yang masuk saat ini hanya ditampilkan sebagai teks notifikasi (e.g., "[Image] caption"). Fitur download media masuk akan hadir di fase berikutnya.
  • Read/Delivery Receipts: Status centang biru belum realtime sepenuhnya (tergantung webhook events).
  • Contact Sync: Nama kontak diambil dari push_name WhatsApp, belum sinkron dengan database CRM kontak lokal.
